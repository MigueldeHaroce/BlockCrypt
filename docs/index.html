BlockCrypt
by MigueldeHaro
Blockcrypt is a password manager designed to unhackable. It does not store passwords locally or on private servers; instead, it implements the use of smart contracts on the blockchain. This decentralized approach makes conventional encryption methods obsolete, providing robust security without relying on centralized storage.
